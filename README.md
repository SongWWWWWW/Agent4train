# Agent4train
### 按照下面这个步骤进行环境配置
```bash
conda create -n agent4train python=3.12
conda activate agent4train
pip install -r requirements.txt
```
```bash
# 请将config.toml.temp更名为config.toml,并将环境变量填入
#启动👇
streamlit run main.py
```
